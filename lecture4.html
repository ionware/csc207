Creating Lists

We are going to look at the Ordered List, The unordered List, and the Definition List

HTML list can be used in structuring Items

Ordered List: Also known as Numbered List, it allows us to create lists which reflect the level of hierarchy. Creating an Ordered List is done using the ol tag.

Each item of the list should be specified with the list item command li.
NOTE: Ordered list uses Arabic numerals as its default numbering type.

Example:

Items required for School

  1. School Fees
  2. Food Stuff
  3. Pocket Money
  4. Books
Attributes of Ordered Lists

The two main attributes for an OL are: type and start. The numbering type can be modified using the type or start attributes in the ol tag.
Values for type attributes are 1, A, a, I or i.
Start attribute specifies at what point to start counting from. The value for the start attribute is usually in Arabic numerals.

Example:

  1. School Fees
  2. Food Stuff
  3. Pocket Money
  4. Books

The start attribute can only take value of numeral. It is used when we want to indicate when our numbering must begin from. By default, ordered list always begin from the first number. Example below

  1. Laptop
  2. Hoodies

Unordered Lists

Another name for the unordered list is Bullet List. We make use of it when the items we want to list doesn't require ordering. To create an unordered list, we make use of the ul tag, which stands for unordered list.
It allows us to create list with various bullet styles. However, those bullets are images and thus do not relect the level of hierarchy except visually.
Example:

  • School Fees
  • Food Stuff
  • Pocket Money
  • Books
Attributes of Unordered Lists

The main attribute for UL is type. The type attribute is used to modify the look of the bullets. The value of type attribute are: disc, circle or square.
Example:

  • School Fees
  • Food Stuff
  • Pocket Money
  • Books

Definition Lists

We use the definition list when we are trying to display a number of terms and associated definition. Allows us to create lists that are more like glossary. We use dl tag to define a term, and it stands for definition list, then followed by dt which stands for definition term, and followed by dd, meaning definition data.
Example:

Computer
Computer is an electronic device capable of taking in inputs, processing this received inputs using set of pre-defined instructions and producing a meaningful output called information.
Hardware
Computer hardware are the components of a computer that can be touch and seen, Example include
  • Mouse
  • Central Processing Unit (CPU)
  • Printer, etc
Software
Software components are the essential part of a computer system which cannot be seen but use.

Nested Lists

Both ordered and unordered Lists can be nested. However, each sublist must be a complete list using the beginning and ending ol and/or the ul tags.
The browser assigns different bullet type to each level of nesting in a nested unordered list. The nested ordered list requires the specification of the numbering type for each level of nesting using the type attribute.
Each level of nesting is indented.
Example:

  1. School Fees
    1. Tuition
    2. Accommodation
  2. Food Stuff
    1. Provision
      • Milk
        • Peak
        • Three Crown
      • Beverages
      • Sugar
    2. Food Stuffs
      • Rice
      • Beans
    3. Oil
  3. Pocket Money
